Estamos na primeira fase do projeto a versão atual é a 1.0

Requisto Colaborador

Foi pedido pelo cliente uma estrutura que pudesse armazenar informações dos colaboradores da embarcação onde esses colaboradores armazenado servirão de dados para a função principal do projeto que é a gestao de "OS" as informações necessária do colaborador são:

MATRICULA
NOME
SOBRENOME
CARGO
SETOR

até o presente momento somente estes dados foram pedidos.

Requisito Centro de Trabalho (ou setor)

Foi pedido neste requisito que fosse armazenado a frente de trabalho que seria responsavel pela execução de uma 'OS' qualquer os dados para armazenamento desta frente são:

NOME
SIGLA
RESPONSAVEL (supervisor)
